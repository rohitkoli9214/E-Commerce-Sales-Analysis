E-Commerce Sales Analysis Project

Project Overview
This project was created using the Olist Brazilian E-Commerce Dataset in Microsoft Excel. The aim of the project is to analyze sales performance, customer reviews, product categories, and regional sales trends by creating an interactive dashboard.

Tools Used
• Microsoft Excel
• Power Query
• Pivot Tables
• Charts and Dashboard

Dataset Files Used
• Orders
• Order Items
• Customers
• Products
• Sellers
• Reviews
• Payments
• Geolocation
• Translation

Questions Solved
1. Which product category has the highest revenue?
Answer: Health & Beauty

2. Which month had peak sales?
Answer: nov-2017

3. Which state performs best?
Answer: São Paulo (SP)

4. What is the average order value trend?
Answer: The average order value remained relatively stable with minor fluctuations.

5. What is the customer review score distribution?
Answer: Most customers gave 5-star ratings.

Business Recommendations
• Increase marketing for Health & Beauty products.
• Focus promotional campaigns in São Paulo.
• Maintain customer service quality to preserve high ratings.
• Prepare inventory for peak sales periods.
• Encourage digital payments as credit cards are the most preferred method.

Key Insights
• Health & Beauty is the highest revenue generating category.
• SP generated the highest revenue among all states.
• Most customers gave 5-star ratings.
• Sales increased significantly during 2017–2018.
• Credit Card is the most preferred payment method.
